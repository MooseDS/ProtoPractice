# Running Web, Mobile (Android, Ios) 

### 목표
Web, Android, IOS 대중적인 분야를 프로토타입으로 구현하면서 개발 역량을 성장시키는 것

## 개발 스펙

### web (NextJs)
- UI - CSS, Tailwind CSS, shadcn/ui
- 상태관리 - useStaet, useEffect, zustand, TanStack Query (api 사용이 가능하다면)
- 서비스 구조 - Router, Form Validation
- 라이브러리 - Chart, Infinite Scroll, Dashboard 등

### android
- UI - Compose, Material UI Custom
- 상태관리 - MVVM, Room, Room - API 동기화 
- 서비스 구조관리 - Navigation, Form Validation
- 라이브러리 - LazyColumn, Chart,  Pagination
